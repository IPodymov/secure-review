# Secure Review — AI-помощник для код-ревью

**Кратко:** Визуализатор изменений (diff) с тепловой картой активности/риск-зон и AI-ассистентом, который объясняет подозрительные фрагменты кода и даёт рекомендации по исправлению.

## Функции (MVP)
- Принимает GitHub репозиторий или PR
- Строит diff + heatmap
- Интеграция SAST инструментов: Semgrep, Bandit
- AI-слой: объясняет проблемы, генерирует отчёт и рекомендации
- Веб UI: интерактивная heatmap

## Технологии
- Backend: Python + FastAPI
- Frontend: React + Tailwind
- Инструменты анализа: Semgrep, Bandit, pip-audit / npm audit
- AI: OpenAI API (только объяснение и сводка)
